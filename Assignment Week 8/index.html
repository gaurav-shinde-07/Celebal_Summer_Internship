Git Commands and Scenarios

1️⃣ Stage and Commit All Changes

git add .
git commit -m "feat: update login and registration form UI/UX and add Firebase integration"

2️⃣ Move Commits to Correct Branch

git checkout -b correct-branch
# commits follow to new branch

git checkout wrong-branch
git reset --hard HEAD~n
# Replace n with number of commits to undo
git push origin wrong-branch --force

3️⃣ Create, Commit, and Push a New Branch

git checkout main
git pull origin main
git checkout -b feature/my-feature
# Make changes
git add .
git commit -m "feat: add new feature"
git push -u origin feature/my-feature

4️⃣ Contribute to an Open Source Project

# Fork → Clone
https://github.com/your-username/project.git

git checkout -b fix/issue-123
# Make changes
git add .
git commit -m "fix: correct layout bug"
git push origin fix/issue-123
# Create PR on GitHub

5️⃣ Resolve Merge Conflicts

git checkout main
git pull origin main
git checkout your-branch
git merge main
# Resolve conflicts manually
# Then:
git add .
git commit

6️⃣ Create Feature Branch from Latest Main

git checkout main
git pull origin main
git checkout -b feature/awesome-update

7️⃣ Revert to a Specific Commit

git log
# Copy the commit hash
git reset --hard <commit-hash>
git push origin HEAD --force

8️⃣ Restore Deleted File from Previous Commit

git log -- <file-path>
git checkout <commit-hash>^ -- <file-path>
git add <file-path>
git commit -m "fix: restore deleted file"
